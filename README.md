🏏 IPL End-to-End Data Analysis Project (2008-2024)
🚀 Project Overview
This project provides a comprehensive analysis of IPL match data from 2008 to 2024. I built an end-to-end data pipeline starting from raw data ingestion in PostgreSQL to creating an interactive dashboard in Power BI.
🛠 Tech Stack
Database: PostgreSQL (SQL)
Data Visualization: Power BI Desktop
Version Control: GitHub
📊 Key Features & Insights
Data Engineering: Cleaned and sanitized 1,000+ match records to handle schema mismatches and delimiter collisions.
Interactive Dashboard: Features a 1,095 total match KPI, team-wise winning trends, and toss decision impacts.
Advanced Visuals: Used dynamic slicers, custom tooltips, and a professional dark theme for better storytelling.
📁 Repository Structure
Bathri_ipl.pbix: The final Power BI dashboard file.
IPL SQL.sql: SQL scripts used for table creation and data cleaning.
matches.csv: Raw IPL dataset.
